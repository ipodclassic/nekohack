NEKOHACK

NEKOHACK — это форк (модифицированная версия) популярного чита ThunderHack.
:rocket: Описание

Этот проект был создан для улучшения функционала оригинального ThunderHack, исправления багов и добавления новых полезных возможностей для комфортной игры.
:sparkles: Новые функции

Я добавил в клиент следующие модули:

    WindJump — позволяет совершать прыжки с использованием механик ветра.

    AutoMace — автоматическое использование булавы (Mace) в бою.

    2DCompanion — теперь в окне компаньона отображаются 2 новые кастомные картинки.

:tools: Установка

    Клонируй репозиторий: git clone https://github.com/ipodclassic/nekohack.git

    Собери проект через Gradle.

    Запускай и наслаждайся игрой!!
